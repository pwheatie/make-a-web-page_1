Lesson One: The Basics of the Web and HTML

World Wide Web

It is a system of interconnected public web pages accessible through the Internet. The links between pages make it web like. Major pieces of the World Wide Web (WWW, W3, the Web) include your computer, the Internet, and servers. Requests are sent by computers through the Internet, those request get filled by servers that send the information back to the computer and use a browser to display the requested page

HTML stands for Hypertext Markup Language. It is structured with elements. Elements are created with opening and closing tags. Each element is created with opening and closing tags: < >. You can provide additional functionality and information with attributes. Attributes have values that increase the power and flexibility of HTML. This is an example of an element with an attribute: <p class="nice">Hello World!</p>. As you can see, between the opening and closing tags we have provided assigned the class nice and added the text Hello World.

Why Computers Are A Bit Dense

Computers interpret code literally. So any mistakes in code large or small can cause major problems in a program.

Lesson Two: HTML and The Structured Document

Treelike Structure of HTML

A web page coded with HTML can be represented by a tree structure or tree diagram. While these representations are really upside down trees, they are helpful in illustrating the nested structure used with HTML. This Wikipedia entry gives a good breakdown of tree diagrams.

White Space

White space is critical for a properly designed web page. Since HTML will line up one element after another, a proper web designer must use white space to break up elements so the page is readable and aesthetically pleasant. A great video on the importance of whitespace can be seen here. There are several great tools devleopers can use to create whitespace on a web page. In-line elements like span can be replaced with block elements like div to create whitespace. In fact, most of an Html page is organized by boxifying content. This allows developers to space the boxed content in a ways that are aesthetically pleasing.

Lesson Three: Color & Flair With CSS

CSS Allows You To Add Style In Various Ways

You can style to your webpage using CSS or Cascading Style Sheets. CSS allows web designers to separate the structure of the page from the styling. Styling can be added using four different techniques:

  • Default style of the browser
  • Stylesheet in a separate file(what you usually use)
  • Stylesheet inside the HTML(this can be used for small projects)
  • Inline style in an element (this can be done but should be avoided)

Boxes, Boxes, and more Boxes!

One important aspect to remember when designing a web page is that an HTML page is constructed using boxes. Those boxes can hold pictures, text, forms, list, and many other elements of a web page. This can help you in the design process. Splitting up content into boxes and then organizing a layout can both simplify and clean up a design. There are four components to boxes:

  • Margin: margin clears an area around the border and is completely transparent
  • Border: goes around the padding and the content, border is inherited from the color property of the box
  • Padding: clears an area around the content/space inside of the box that protects the content
  • Content: this is the image or text on the website, it is the core, or center of the box

Stop Repeating Yourself!!

One of the more powerful concepts about software development is the ability to avoid repetition. You can use CSS to assign rules and classes to HTML tags. This allows you to style large groups of HTML ONCE so you don't have to apply styling to each HTML element separately. This saves time, increases efficiency, while helping keep you keep your code clean and readable.

Code/Test/Refine

As you build your design it is important to focus on the HTML first by looking for repeated styles and semantic elements.Once you write your HTML add your CSS by focusing on the largest elements first. Once you have the styling done, fix the small issues. Using developer tools can really help speed up the process.

Basic HTML Vocabulary

Tag- HTML tag is always contained by brackets
Element- and HTML element refers to an opening tag and closing tag with items like attributes and text between the two tags.
Attribute- is a property of an HTML element. href is an example.
Boxify- The process of boxing content in order to increase flexibility and enhance visual presentation.
